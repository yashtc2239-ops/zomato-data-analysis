# Zomato Restaurant Data Analysis
End-to-End Data Analytics Portfolio Project

**Author:** Yash Chavan | VIT Pune CSE-DS | 2024-28
**Tools:** Python | SQL | Power BI | Machine Learning

---

## Project Overview
Comprehensive analysis of 41,410 Bengaluru restaurants
to identify market expansion opportunities, rating
drivers and business insights for Zomato strategy team.

---

## Key Findings
- Lavelle Road = #1 expansion target (Score 95.6/100)
- Random Forest predicts ratings at R2 = 0.82
- Customer votes drive 69% of rating prediction
- Table booking restaurants rate 0.52 points higher
- Only 1.4% restaurants achieve Excellent rating

## ML Model Results
- Algorithm: Random Forest Regressor
- R2 Score: 0.8157 (81.6% accuracy)
- RMSE: 0.1885 rating points
- Training: 33,128 restaurants
- Testing: 8,282 restaurants
- Top predictor: Customer votes (69.3%)

---

## Business Recommendations
1. Expand in Lavelle Road, Church Street, St. Marks Road
2. Weight engagement metrics in restaurant ranking
3. Target Modern Indian and European cuisine gaps
4. Scale table booking program (2.6x revenue potential)
5. Quality improvement for bottom 10 locations
